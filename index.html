AASP: Archaeological Assertion and Synthesis Profile

Preferred prefix: aasp

Namespace: https://w3id.org/aasp/

Current version: https://w3id.org/aasp/0.4.2/

AASP is a provisional, versioned Linked Data application profile for coordinating existing standards around the representation of revisable archaeological field-survey claims. It keeps evidence states, field procedures, analytical decisions, claims, claim relations, and dataset releases explicitly connected.
