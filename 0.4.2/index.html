AASP 0.4.2

Archaeological Assertion and Synthesis Profile

AASP 0.4.2 is a provisional, versioned Linked Data application profile for coordinating existing standards around the representation of revisable archaeological field-survey claims.

This version focuses on keeping evidence states, field procedures, analytical decisions, claims, claim relations, and dataset releases explicitly connected.

Status

AASP is a research application profile. It does not replace CIDOC CRM, CIDOC CRM extensions, PROV-O, DCAT, SKOS, PeriodO, GeoSPARQL, or CRMgeo. Instead, it coordinates selected relations across these standards for the limited purpose of making archaeological survey syntheses traceable, comparable, rerunnable, and non-destructively revisable.
